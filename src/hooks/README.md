# `/src/hooks/`

This folder contains reusable, global custom React hooks that can be shared across multiple features.

Examples:
- `useLocalStorage`: Helper for reading/writing local storage state.
- `useMediaQuery`: Hook for checking viewport boundaries (desktop/mobile).
- `useDebounce`: Debouncing inputs (e.g., search text).
- `useOnClickOutside`: Closing overlays when clicking outside them.
