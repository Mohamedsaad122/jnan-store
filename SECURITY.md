# Security Policy

## Supported Versions

We actively monitor and patch security vulnerabilities in the following versions of Jnan Store:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please **do not open a public issue**. Instead, report it privately to our security team.

### Steps to Report

1. Email your report directly to **security@jnan-store.com**.
2. Include a detailed description of the vulnerability, the potential impact, and clear step-by-step instructions to reproduce it (or a proof-of-concept script).
3. Our security response team will acknowledge receipt of your report within 48 hours and work with you to analyze and resolve the issue.

We follow responsible disclosure practices and ask that you give us reasonable time to deploy a patch before disclosing any vulnerability details publicly.
